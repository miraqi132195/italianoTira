# Italiano Tira

Premium bilingual (Hebrew / Arabic) restaurant website for **Italiano** in Tira.

## Stack

- React + Vite
- Tailwind CSS v4
- React Router

## Run locally

```bash
npm install
npm run dev
```

## Edit content

All editable content lives in `src/data/`:

| File | Contents |
|------|----------|
| `restaurant.js` | Phone, links, address, hours, gallery images |
| `menu.js` | Categories and menu items |
| `translations.js` | All UI text (Hebrew & Arabic) |

Replace `/public/logo.svg` with your branding logo.

## Build

```bash
npm run build
npm run preview
```
